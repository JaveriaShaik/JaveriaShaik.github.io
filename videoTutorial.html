How to Embed a YouTube Video into Your Web Page

This step-by-step tutorial will teach you how to embed a YouTube video into your own website using HTML. Each step is illustrated with a helpful screenshot.

Step 1: Go to the YouTube Video and click on share button

Open the YouTube video you want to embed. For this example, weâ€™ll use a video titled "HTML & CSS - How to Embed a YouTube Video in Your Website". Click the share button under the video.

Step 2: Select the Embed Option

When you click on share, it will open sharing options. Click the first option Embed.

Step 3: Copy the Embed Code

When you click on Embed, it will show you the HTML code needed to embed the video. Click the Copy button to copy the embed code to your clipboard.

Step 4: Paste the Code into Your HTML Page

Open your HTML file and paste the embed code where you want the video to appear.

Step 5: Create a div and add a class to add modifications to your embed video.

Create a div to make changes to the width and height. refer the image below. Then move your embed code into the div. Change height and width accordingly. In this example we changed the width and height to 100%.

Step 6: Create a css file

Create the class in the css file to set the width and height of the video.

Step 7: Use the website to get the perfect dimentions

If you are not sure what are the perfect dimentions for the video, use can use the website aspectratiocalculator.com. When you type in the width, it'll automatically give you the height.
